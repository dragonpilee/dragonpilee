# Sandra Anna Joshy - Professional Portfolio

[![Netlify Status](https://api.netlify.com/api/v1/badges/your-netlify-id/deploy-status)](https://app.netlify.com/sites/your-netlify-site/deploys)

A high-end, **MAANG-optimized** corporate portfolio for **Sandra Anna Joshy**, Technical Support Lead. Built with **Astro** and **Tailwind CSS**, this profile is designed for maximum impact, speed, and recruiter skimmability.

## 🚀 Key Features

- **MAANG-Ready Experience**: Quantified impact-first bullets (Action-Context-Result pattern).
- **Categorized Expertise**: Technical stack grouped into functional domains (IAM, Infrastructure, Ops).
- **Enterprise UI**: Sticky professional navigation, premium corporate-savvy aesthetic, and a smooth 'Bright/Dark' mode toggle.
- **Recruiter SEO**: Optimized meta-tags and schema for search engine and internal recruiter visibility.
- **Netlify Optimized**: Automated asset processing and security headers via `netlify.toml`.

## 🛠️ Tech Stack

- **Framework**: [Astro 5](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: [Netlify](https://www.netlify.com/)
- **Icons**: Lucide/SVG Custom

## 📂 Project Structure

```text
/
├── src/
│   ├── components/  # Atomic corporate UI components
│   ├── layouts/     # Base layout with theme toggle & SEO
│   ├── pages/       # Astro pages
│   └── styles/      # Global theme-aware design system
├── public/          # Static assets
└── netlify.toml     # Deployment optimization
```

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## 🌐 Deployment

This site is optimized for **Netlify**. Simply connect this repository to your Netlify account, and it will auto-detect the Astro configuration.
